Instructions for use of this integer factorizer:

Return to the Math APIs page.

  • After ...herokuapp.com above you should type /json/ followed by your natural number. The factors property of the json output will be an array of 2-element arrays. The zeroth element is a prime factor, and the other element is the power to which it is raised, when forming the prime factorization. If you want the response to be html rather than json, simply omit /json from the address. E-notation is not allowed. Your number must be less than 263 (which approximately equals "9" followed by 18 zeros).
  • examples:
    • 359 is prime.
    • 360 = 23·32·5
