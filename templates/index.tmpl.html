Instructions for use of this factorizer:

Return to the Math APIs page.

  • General:
    • After ...herokuapp.com above you should type /json/ followed by your particular input. Spaces are allowed - but discouraged - because %20 will replace each space after you hit return, thereby making the address uglier. E-notation is not allowed. All numbers must be natural numbers less than 263 (which approximately equals "9" followed by 18 zeros).
    • If you want the response to be html rather than json, simply omit /json from the address.
    • greatest common divisor (GCD):
      • Your input should be a bracket-enclosed list of one or more comma-separated numbers.
      • examples:
        inputresult
        [9,12]3
        [9,10]1
        [13]13
        [8,10,12]2
    • factorization of a real integer:
      • Your input should simply be a number.
      • The factors property of the json output will be an array of 2-element arrays. The zeroth element is a prime factor, and the other element is the power to which it is raised, when forming the prime factorization.
      • examples:
        inputresult
        359prime
        36023·32·5
